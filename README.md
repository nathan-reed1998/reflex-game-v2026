## What is The Reflex Game?

The Reflex Game is a lightweight typing challenge that runs in the browser and is built entirely with HTML. It keeps the experience simple: quick reactions, minimal friction, and a game flow that can be launched without any complicated setup.

It was built for Hack Club's Summer Of Making Hackathon, so the project has a clear event-specific origin and a narrow, focused goal. If you want a compact typing game that works directly on the web, this is a straightforward fit.
